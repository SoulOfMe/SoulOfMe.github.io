## 常见问题

以下是一些关于本项目的常见问题及其解答。

### 1. 这个项目是免费的吗？

是的，本项目完全免费，并采用 MIT 许可证开源。您可以自由使用、修改和分发。

### 2. 我需要了解 Jekyll 才能使用吗？

基本不需要。如果您只是想修改内容，只需编辑 Markdown 文件（`.md`）即可。如果您想进行更深入的自定义，了解一些 Jekyll 的基础知识会很有帮助。

### 3. 如何部署到 GitHub Pages？

只需将您的代码推送到名为 `username.github.io` 的仓库，或者在任何仓库的 `gh-pages` 分支上即可。GitHub Pages 会自动为您构建和部署网站。

### 4. 为什么选择 Tailwind CSS？

Tailwind CSS 是一个功能强大的“原子化 CSS”框架，它能让您在不离开 HTML 的情况下快速构建现代设计。通过 CDN 使用，可以免去复杂的构建过程。

### 5. 我可以添加自己的 CSS 吗？

当然可以。您可以在 `_layouts/default.html` 的 `<head>` 部分添加自己的样式表链接，或者直接编写内联样式。
